# 摩托车成本模型

一个用于可视化摩托车相关保险成本、利润与关键指标的前端小工具。通过调节输入参数，可即时看到车险与摩意险的成本率、边际贡献和盈利表现。

## 功能概览
- **快速方案切换**：预置 A–D 四套方案，切换即可对比不同赔付率假设下的表现。
- **实时可视化**：车险、摩意险及合计视角的金额与比率图表会随参数即时刷新。
- **关键指标提示**：顶部 KPI 和智能洞察文本突出盈利情况与需要关注的指标。
- **数据导出**：一键将当前测算结果导出为 CSV，便于留存与二次分析。
- **浅/深色主题**：支持主题切换并自动记忆上次选择。

## 快速运行
1. 在本地启动一个静态文件服务器（任选其一）：
   - 使用 Python：`python3 -m http.server 8000`
   - 使用 Node.js：`npx serve .`
2. 在浏览器访问 `http://localhost:8000`（或服务器对应端口）。
3. 切换方案或调整右侧参数抽屉中的输入，观察 KPI 与图表的即时变化。

## 使用提示
- 导出数据时会在文件名中附带日期与序号，便于区分多次测算结果。
- 若需要恢复默认参数，可点击方案 C（保本）按钮重新应用基准配置。
- 图表颜色会根据主题和指标表现自动调整，负向指标会高亮提示以便识别风险。

## 回归测试建议（最小集）
- **主题切换**：在顶部点击主题按钮两次，确认浅/深色样式与主题标签同步更新并持久化刷新后状态。
- **方案切换**：在方案按钮或抽屉中切换 A–D 方案，观察参数输入框、方案标签及高亮提示正常刷新。
- **参数应用**：修改任意输入，验证校验提示与“应用参数”后 KPI、洞察文案与瀑布图同步更新。
- **图表更新**：切换分析标签（车险/摩意/合计），确认对应图表渲染并在窗口缩放后能正常自适应。
